# Generic web app
Generic web app.
## How to use
Clone a repository and run `npm run deploy` from `src/server/` in order to build the app, and then `npm start` to run a server locally.
## Technologies used:
* DB
    * MariaDB
    * Join monster - graphQL to SQL
* Dev tools
    * Webpack
    * Babel
    * Eslint
    * Testing tools
        * Mocha
        * Chai
        * Supertest
        * Sinon
* Frontend
    * Pug
    * SASS
    * Bootstrap
* Web frontend
    * AngularJS
